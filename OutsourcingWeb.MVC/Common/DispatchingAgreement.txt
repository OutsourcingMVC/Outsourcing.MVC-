font-family: 宋体; font-size: 16px;" > &nbsp; &nbsp;&nbsp; &nbsp; &nbsp; 往甲方工作的乙方人员，甲方每聘用一名，必须向乙方支付该人员两个月的外包费用，作为补偿。</span></p><p style = "margin: 0px 0px 0px 56px; line-height: 150%; -ms-layout-grid-mode: char;" ><span style = "line-height: 150%; font-family: 宋体; font-size: 16px;" > 3.4.1 项目开发期间，甲方如向乙方提出缩减人员的要求，需提前1个月通知乙方，否则乙方不得以任何理由 </span></p><p style = "margin: 0px 0px 0px 70px; line-height: 150%; -ms-layout-grid-mode: char;" ><span style = "line-height: 150%; font-family: 宋体; font-size: 16px;" > &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; 进行人员缩减。</span></p><p style = "margin: 0px 0px 0px 70px; line-height: 150%; -ms-layout-grid-mode: char;" ><span style = "color: black; &nbsp; &nbsp; 保密 </span></h1><p style = "margin: 0px 0px 0px 56px; line-height: 150%;" ><span style = "line-height: 150%; font-family: 宋体; font-size: 16px;" > 5.1 &nbsp; 任一方不得向第三方透露本合同内容。</span></p><p style = "margin: 0px 0px 0px 56px; font-size: 16px;" > &nbsp; &nbsp; &nbsp; 不意味着任何所有权、专利权的转让。</span></p><p style = "margin: 0px 0px 0px 56px; line-height: 150%;" ><span style = "line-height: 150%; font-family: 宋体; font-size: 16px;" > 5.4 &nbsp; 在本合同届满或终止后，乙方人员应归还甲方提供的所有技术资料或文件等，并承诺不保留任何副本。</span></p><p style = "margin: 0px 0px 0px 56px; line-height: 150%;" ><span style = "line-height: 150%; font-family: 宋体; font-size: 16px;" > 5.5 &nbsp; 双方理解，违反本条规定将可能给对方造成难以弥补的损失，为此同意承担由此产生的经济上和法律上 </span></p><p style = "margin: 0px 0px 0px 56px; line-height: 150%;" ><span style = "line-height: 150%; font-family: 宋体; font-size: 16px;" > &nbsp; &nbsp; &nbsp; &nbsp; 的责任。</span></p><p style = "margin: 0px 0px 0px 56px; line-height: 150%;" ><span style = "line-height: 150%; </p><h1><span style = "font-family: 宋体;" > 第六条 &nbsp; &nbsp; &nbsp; 违约责任 </span></h1><p style = "margin: 0px 0px 0px 56px; line-height: 150%; -ms-layout-grid-mode: char;" ><span style = "line-height: 150%; font-family: 宋体; font-size:16px;" > &nbsp; &nbsp; &nbsp; 本合同，停止支付余款，并按照非乙方人员数量每人贰仟元人民币赔偿甲方，赔偿最大金额贰万元人民币。</span></p><p style = "margin: 0px 0px 0px 56px; line-height: 150%; -ms-layout-grid-mode: char;" ><span style = "line-height: 150%; -ms-layout-grid-mode: char;" ><span style = "line-height: 150%; font-family: 宋体; font-size: 16px;" > &nbsp; &nbsp; &nbsp; &nbsp;&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; 用，乙方按照法律流程，律师费、诉讼费差旅费等各项费用由甲方全部承担。</span></p><p style = "line-height: 150%; text-indent: 16px; -ms-layout-grid-mode: char;" ><span style = "line-height: 150%; font-family: 宋体; font-size: 16px;" > &nbsp; &nbsp;&nbsp; &nbsp; 6.3 &nbsp; 乙方违反本合同约定的知识产权条款，甲方有权要求乙方赔偿甲方因此所受全部损失。</span></p><h1><span style="font-family: 宋体;">第七条&nbsp;&nbsp;&nbsp; 合同变更</span></h1><p style="margin: 0px 0px 0px 56px; line-height: 150%;"><span style="line-height: 150%; font-family: 宋体; font-size: 16px;">7.1&nbsp;任何一方如需要对合同内容作修改或补充，须提出变更理由和变更方案，经双方协商一致，由双方授权</span></p><p style="margin: 0px 0px 0px 56px; </p><h1><span style="font-family: 宋体;">第八条&nbsp;&nbsp;&nbsp; 争议解决</span></h1><p style="margin: 0px 0px 0px 56px; line-height: 150%;"><span style="line-height: 150%; font-family: 宋体; font-size: 16px;">8.1&nbsp;本合同及其修订本的有效性、履行和与本合同及其修订本效力有关的所有事宜，将受中华人民共和国法</span></p><p style="margin: 0px 0px 0px 56px; line-height: 150%;"><span style="line-height: 150%; font-family: 宋体; font-size: 16px;">&nbsp;&nbsp;&nbsp;&nbsp;律管辖，任何争议仅适用中华人民共和国法律。</span></p><p style="margin: 0px 0px 0px 56px; line-height: 150%;"><span style="line-height: 150%; font-family: 宋体; font-size: 16px;">8.2&nbsp;甲乙双方因合同的解释或履行发生争议时，首先应争取通过友好协商解决。</span></p><p style="margin: 0px 0px 0px 56px; line-height: 150%;"><span style="line-height: 150%; font-family: 宋体; font-size: 16px;">8.3&nbsp;如协商不能解决时，合同的任何一方可将争议提交北京仲裁委员会，并按该委员会的仲裁规则进行仲裁。</span></p><p style="margin: 0px 0px 0px 56px; line-height: 150%;"><span style="line-height: 150%; font-family: 宋体; font-size: 16px;">&nbsp;&nbsp;&nbsp;&nbsp;仲裁裁决对双方都有约束力。</span>
</p><p style="margin: 0px 0px 0px 56px; line-height: 150%;"><span style="line-height: 150%; font-family: 宋体; font-size: 16px;">8.4&nbsp;争议进行仲裁期间，除争议事项外，甲乙双方应继续履行各自本合同中规定的义务和行使权利。</span></p><p><strong><span style="line-height: 150%; font-family: 宋体; font-size: 16px;"><br/></span></strong></p><h1><span style="font-family: 宋体;">第九条&nbsp;&nbsp;&nbsp; 不可抗力</span></h1><p style="margin: 0px 0px 0px 50px; line-height: 150%;"><span style="line-height: 150%; font-family: 宋体; font-size: 16px;">9.1 不可抗力是指本合同生效后，发生不能预见并且对其发生和后果不能防止或避免的事件，如地震、台风、水</span></p><p style="margin: 0px 0px 0px 50px; line-height: 150%;"><span style="line-height: 150%; font-family: 宋体; font-size: 16px;">&nbsp;&nbsp;&nbsp; 灾、火灾、战争等，致使直接影响本合同的履行或不能按约定的条件履行。</span></p><p style="margin: 0px 0px 0px 50px; line-height: 150%;"><span style="line-height: 150%; font-family: 宋体; font-size: 16px;">9.2 发生不可抗力的一方应立即通知对方，并在十五天内提供不可抗力的详情及将有关证明文件送交对方。</span></p><p style="color: rgb(0, 0, 0); line-height: 150%; font-style: normal; font-weight: normal; margin-top: 0px; margin-bottom: 0px;"><span style="line-height: 150%; font-family: 宋体; font-size: 16px;">&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;a.发生不可抗力事件时，甲乙双方应协商以寻找一个合理的解决方法，并尽一切努力减轻不可抗力产生的</span>
</p><p style="color: rgb(0, 0, 0); line-height: 150%; font-style: normal; font-weight: normal; margin-top: 0px; margin-bottom: 0px;"><span style="line-height: 150%; font-family: 宋体; font-size: 16px;">&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; 后果。</span></p><p style="color: rgb(0, 0, 0); line-height: 150%; font-family:;"><span style="line-height: 150%; font-family: 宋体; font-size: 16px;">&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;b.如不可抗力事件持续三十天时，甲乙双方应友好协商解决本合同是否继续履行或终止的问题。</span></p><h1><span style="font-family: 宋体;">第十条&nbsp;&nbsp;&nbsp; 其他</span></h1><p style="margin: 0px 0px 0px 50px; line-height: 150%;"><span style="line-height: 150%; font-family: 宋体; font-size: 16px;">10.1 本合同一式贰份，甲乙双方各持壹份，每份具有同等法律效力。</span>
